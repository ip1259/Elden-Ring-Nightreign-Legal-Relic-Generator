# Elden-Ring-Nightreign-Legal-Relic-Generator
This repo is used to verify whether combinations of Relic effects are valid.<br>
<br><b>Please use the following link to create your own copy.</b>
<br><b>https://docs.google.com/spreadsheets/d/1s_HFEHD5WRSlKXd5i1vFLL_kdQ_5mYz0HsweqhCa7Qg/edit?usp=sharing</b>

## Main Sheet: sheets.xlsx
所有操作在第一頁"UI"頁操作<br>
不要修改其他頁，除非你知道你在做什麼<br>
UI頁分兩個部分A1:B8、A11:B18<br>
請忽略D、E兩欄他們是排序計算用的<br>

All operations should be performed on the first page, "UI".<br>
Do not modify other pages unless you know what you are doing.<br>
The UI page is divided into two sections: A1:B8 and A11:B18.<br>
Please ignore columns D and E; they are used for sorting calculations.<br>

### UI
你可以操作的部分是UI頁的B1:B8<br>
A欄是標籤，請不要隨意更動<br>
B1:語言<br>
B2:Relic的ID-名稱(顏色)<br>
B3-B5:正面效果，請依序選擇，衝突的效果會在選擇後自動排除<br>
B6-B8:詛咒效果，只有深夜遺物會有<br>

1.0.3版的效果池會出現在200-235，這個區間是代表由商店購買的，<br>
100-135同樣是商店購買但效果池會是1.0.2版以前的，<br>
未購買DLC請不要使用200-235區間的Relic編號，<br>
因為理論上來說你們不可能會有，<br>
我強烈建議只用100-235這區間的RelicID<br>


The section you can edit is B1:B8 on the UI page.<br>
Column A contains labels; please do not modify them arbitrarily.<br>
B1: Language<br>
B2: Relic ID - Name (Color)<br>
B3-B5: Positive Effects - Select in sequence; conflicting effects will be automatically excluded upon selection<br>
B6-B8: Curse Effects - Only available on Deep Relics<br>

The effect pool for version 1.0.3 appears in the 200-235 range.<br>
This range represents items purchased from the store.<br>
The 100-135 range is also store-purchased, but the effect pool will be from versions prior to 1.0.2.<br>
If you have not purchased the DLC, please do not use Relic IDs from the 200-235 range, as theoretically you should not possess them.<br>
I strongly recommend using only Relic IDs from the 100-235 range.<br>

### Final Result
A11:B18是最終結果區，包含排序及補上空ID(Empty ID:4294967295)<br>
請注意，你的選擇順序跟最終順序可能有所不同<br>

A11:B18 is the final results range, containing sorted data and filled-in empty IDs (Empty ID: 4294967295).<br>
Please note that your selection order may differ from the final order.<br>
