# Elden-Ring-Nightreign-Legal-Relic-Generator
This repo is used to verify whether combinations of Relic effects are valid.

## Main Sheet: sheets.xlsx
所有操作在第一頁"UI"頁操作
不要修改其他頁，除非你知道你在做什麼
UI頁分兩個部分A1:B8、A11:B18
請忽略D、E兩欄他們是排序計算用的

All operations should be performed on the first page, "UI".
Do not modify other pages unless you know what you are doing.
The UI page is divided into two sections: A1:B8 and A11:B18.
Please ignore columns D and E; they are used for sorting calculations.

## UI
你可以操作的部分是UI頁的B1:B8
A欄是標籤，請不要隨意更動
B1:語言
B2:Relic的ID-名稱(顏色)
B3-B5:正面效果，請依序選擇，衝突的效果會在選擇後自動排除
B6-B8:詛咒效果，只有深夜遺物會有

1.0.3版的效果池會出現在200-235，這個區間是代表由商店購買的，
100-135同樣是商店購買但效果池會是1.0.2版以前的，
未購買DLC請不要使用200-235區間的Relic編號，
因為理論上來說你們不可能會有，
我強烈建議只用100-235這區間的RelicID

The section you can edit is B1:B8 on the UI page.
Column A contains labels; please do not modify them arbitrarily.
B1: Language
B2: Relic ID - Name (Color)
B3-B5: Positive Effects - Select in sequence; conflicting effects will be automatically excluded upon selection
B6-B8: Curse Effects - Only available on Deep Relics

The effect pool for version 1.0.3 appears in the 200-235 range.
This range represents items purchased from the store.
The 100-135 range is also store-purchased, but the effect pool will be from versions prior to 1.0.2.
If you have not purchased the DLC, please do not use Relic IDs from the 200-235 range, as theoretically you should not possess them.
I strongly recommend using only Relic IDs from the 100-235 range.
